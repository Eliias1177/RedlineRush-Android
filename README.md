# Redline Rush: Vertical Shift - Frontend

Aplicación móvil nativa de carreras inspirada en mecánicas de CSR Racing, desarrollada con **Kotlin** y **Jetpack Compose**.

##  Autor
* **Jesús Elías Arriaga Salinas** - *Desarrollo de Físicas, UI y Cliente de Red*

## Institución
**Universidad Politécnica de la Región Ribereña**

## Tecnologías
* Kotlin / Jetpack Compose
* SignalR (Real-time Multiplayer)
* Retrofit (API Rest)
